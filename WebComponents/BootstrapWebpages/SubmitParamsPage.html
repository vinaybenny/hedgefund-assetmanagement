General parameters Select the following parameter values for running your simulation, or choose Submit to run it for the default values.

Investor's wealth at the beginning of the simulation time-frame.

Risk Aversion for the investor. Should be greater than 1.

The time duration for the investment.

Expected rate of interest for an absolutely risk-free investment.

The risk of a decline in the value of a security.

If investor's total wealth falls below this percentage of initial wealth, the risky positions are closed.

Maximum loan the investor can avail, as a multiple of initial wealth.

Mutual fund parameters Following are the simulation parameters specific to mutual funds.

Management fee for mutual funds, expressed as a percentage on initial wealth.

Index Fund parameters Following are the simulation parameters specific to stock market index.

Simulation-specific parameters Following are parameters that determine the simulation accuracy and duration. Note that we use Monte-Carlo simulation framework. A large number of possible stock market trajectories are created that reflect the randomness of the market. Larger the number of such trajectories, more accurate will be the outcome and slower will be the simulation.

Intervals of a year for which the wealth value is being monitored. For example, a value of 4 implies quarterly monitoring, whereas 365 implies daily monitoring. Higher values increase accuracy and time taken for simulation.

Number of stock market trajectories to be simulated. Higher values increase accuracy and time taken for simulation.
